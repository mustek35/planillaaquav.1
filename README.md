# PlanillaAquaV

This project requires certain environment variables for database and SFTP connections.

## Required environment variables

- `DB_HOST`
- `DB_PORT`
- `DB_NAME`
- `DB_USER`
- `DB_PASS`
- `SFTP_HOST`
- `SFTP_PORT`
- `SFTP_USER`
- `SFTP_PASS`

Ensure these variables are set before running the application.
